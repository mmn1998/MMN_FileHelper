#### Dedicated to all Iranians and friends of Iran

- Upload/Download/Working with files in **.NET8**
- Export Excel & CSV in **.NET8**
#### With this Features
> - Export Excel & CSV from Generic IEnumerable of a model
> - Upload file With Base64 string and IFormFile and Get byte[] from Base64 string
> - Content-Type (Mime-Type) Mapping to related file extensions
> - Identifying the true content of a file (Mime-Type) even if its extension has changed (For a limited number of files contained in the code)
